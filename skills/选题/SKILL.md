# 选题引擎

为迪迪（AI工具/创业号，build-in-public）每天产出打好分的选题清单。

## 必读（每次先读）
1. `账号北极星.md` —— 定位、受众、红线、北极星指标（涨粉千播+完播率，不看播放）
2. `爆款知识库/套路库.md` —— §1 九类选题公式、§7 评分卡、§8 迪迪独家结合点

## 流程
1. **拉热点（多路，一手优先）**：
   - **一手信息源**（★优先，最快最准）：`python3 scripts/firsthand.py` —— GitHub 新热 repo（什么工具/agent/skill 正在火，最贴 build-in-public）+ HN 近7天 AI 热议（技术圈原始观点）。想深挖某条 → WebFetch 它的官方博客/论文/repo README 原文（一手）。
   - **AI 行业热点**：调 aihot skill —— 二手聚合，覆盖广、补一手没覆盖的。与一手**交叉验证**：两边都冒头=强信号（如 GLM-5.2）。
   - **抖音平台热榜**（辅）：`python3 scripts/douyin_hot.py` —— AI 过滤；多数日子无 AI 会提示跳过；AI 杀进热榜=破圈强信号 / 借热梗句式(`--all`)。
   - **赛道热点**：`对标监控/待拆解-*.md` —— 对标新爆款在火什么。
2. **（可选）看自家数据**：近期复盘卡 / `loop_review.py` 结果，避开刚扑/限流的方向。
3. **生成候选**：按套路库 §1 的九类公式，结合 aihot 热点 + 迪迪独家结合点（§8），出 8-10 个候选选题。
3.5 **验证/挖需求（`last30days` skill·按需深挖层）**：对评分靠前的 2-3 个候选，调 `last30days` 跨平台扒近 30 天真实讨论——
   - 用途：①验证"这题到底有没有人在讨论、热度真假"②挖评论区/帖子里的真实痛点和争议点，当文案素材。
   - 与发现层的分工：firsthand/aihot/抖音热榜=每天 push 发现新题；`last30days`=选定候选后**手动**按需深挖（它慢，并行抓取+合成，不进每日自动拉）。
   - 免费源够用：Reddit/HN/GitHub/Polymarket/Web；X/YouTube/TikTok 要额外配置，先不依赖。⚠️ 只喂选题关键词，别存敏感数据。
   - 把验证结论（讨论度真假/痛点摘录）回填到候选的"凭什么爆"和钩子里。
4. **评分卡打分**（每项 0-2，满分 10）：北极星匹配 / 收藏属性 / 迪迪独家性 / 钩子强度 / 时效热点。
5. **排序判定**：≥8 优先做、6-7 备选、<6 弃。每条标明分项得分 + 凭什么爆 + 建议钩子 + **📎原始信息源（必附，可溯源）**——一手优先（GitHub repo / 官方博客 / 论文原文 / 智谱等官网），用了二手（媒体/X转述）必须标注"二手"并建议出片前回一手核；内部依据（自己的视频/套路库）注明"内部·无外链"。
6. **落文件**：写入 `选题清单/<日期>-第N批.md`。
7. **👤 人工卡点**：选哪条由用户拍板，**不自动进文案环节**。

## 铁律
- 收藏率思维：优先"高收藏型"选题（精准粉），避开高分享低收藏的泛流量题。
- 避红线：不出变美内容、不出"教你赚钱"割韭菜调性、不出灰产题（见套路库 §6）。
- 优先级：build-in-public(A) > 垂直合集(B) > 场景刚需(C) > 变现实操(D) > 转型叙事(F)。
- 蹭活动扶持位：标签带 #vibecoding大赏 #青年创作者成长计划 等。
