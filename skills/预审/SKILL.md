# 发布前预审

对定稿文案 + 标题 + tag 做一遍风险扫描，**发布前最后一道人工卡点的助手**。

## 红线检查清单

**🔴 限流/仅自己可见诱因（最高优先）**
- 文案里贴提示词全文 / 口令 / 微信号 / 联系方式 / 外部链接 / 二维码 → 改成"评论区扣【XX】我发你"
- 引导站外（"去我主页链接""加我微信"明示）→ 含蓄化

**🔴 标签 / 定位禁区**
- 变美类标签或内容（#AI变美 等）→ 删（北极星已转型，会污染工具号标签）
- 割韭菜词：月入过万 / 躺赚 / 包教包会 / 三天速成 / 财富自由 → 删或改"我真做出来了"的实证调性
- 灰产 / 搬运 / 教人钻空子的调性 → 转正向

**🟡 违规/夸大词**
- 绝对化：最/第一/唯一/100%（广告法）
- 医疗、功效、投资收益承诺
- 标题党但正文无干货

**🟢 正向检查**
- 钩子是否结果先行？结尾有无认知金句？有无文档钩子导私域？标题↔钩子是否呼应？

## 输出
逐条列：🔴必改 / 🟡建议改 / 🟢OK，每条给具体改法。最后一句总结："可发 / 改完可发 / 不建议发"。
