# 迪迪 视频模板库

> 多风格模板库。用户分享好风格 → 我快速拆成条目 → 验证好就沉淀 → 未来按内容/调性快速匹配复用。

## 状态图例
🌱 草拆(刚拆，未验证) · 🧪 试用中 · ✅ 已沉淀(验证好，放心复用) · ❌ 弃用

## 文件
- `编排方案模板.md` — **录前**逐段规划模板（展示方式/风格/素材分工/全程常驻）
- `幻灯片风格.md` — PPT / 数据图 / 框架图 的视觉风格
- `画面与动效.md` — 视频合成画面、字幕、转场、动效
- `博主拆解.md` — 对标博主完整风格画像

## 🔧 拆模板工作流（你分享 → 我沉淀）
1. **你分享**：PPT 截图 / Remotion 博主视频 / 链接
2. **我快速拆**：抽帧读图 → 提炼[配色 / 字体 / 版式 / 标志元素] → 写成条目(🌱草拆) + 出一张样张到 `~/Desktop/迪迪视频预览/`
3. **你验收样张**：效果好 → 升 ✅ 已沉淀；不好 → 迭代或 ❌ 弃用
4. **复用**：做新片时按内容/调性从库里挑风格匹配

## 🔁 每条视频做完的复盘（必做）
1. **记录错误**：返工 / 审美翻车 / 渲染失败 / 踩的坑 → 写进该片 `projects/<名>/复盘.md`；通用教训 → 长期记忆
2. **报告用户**：这条 **沉淀了哪些**(新模板 / 新校正 / 新偏好) + **模板库新增哪些条目**
3. **升级**：把验证好的图形升级为可复用 Remotion 组件

## 条目 schema（每个模板都按这个写）
```
### [编号] 名称 — 状态
- 来源：谁 / 哪条 / 链接
- 适用：什么场景/调性用它
- 配色：色值
- 字体：标题 / 正文 / 数字
- 版式·标志元素：一眼认出它的特征
- 实现：Remotion / PIL 怎么落地
- 样张：~/Desktop/迪迪视频预览/xxx.png
```

## 怎么选风格匹配一条片
干货/教程 → 亮底编辑撞色(权威+清晰)；轻松/build-in-public → 掺活泼涂鸦；
画面统一走暗场 HUD 合成，幻灯片用亮底**反差**压上去。
