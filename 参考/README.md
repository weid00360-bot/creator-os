# 参考 · 平台官方规则 + 学习沉淀

> 跟 `爆款知识库`（拆别人视频）区分开：这里放**平台官方规则、方法论、我的学习思考**。
> 用户发来的官方指南/审核文档、我从中提炼的可复用结论，都沉淀在这。

## 目录约定

| 文件 | 放什么 | 喂给谁 |
|---|---|---|
| `抖音优质内容指南.md` | 抖音官方说"奖励什么内容"的提炼 | 套路库校准 / 选题 / 文案 |
| `抖音审核红线.md` | 抖音审核方向、违规红线（官方口径） | 「预审」skill |
| `人味儿写作.md` | 写/改稿保住存在感的方法论（位置/代价/手迹 + AI味事后清单） | 「文案」skill |
| （后续按需加） | 其他平台规则 / 方法论学习 | 对应环节 |

## 原则
- **官方口径 > 我的经验**：官方规则和我之前凭经验总结的（如套路库§6红线）冲突时，以官方为准，并回头订正套路库。
- 每份注明**来源链接 + 获取日期**（平台规则会变，过期要复核）。
- 提炼成"可执行结论"，不堆原文。
