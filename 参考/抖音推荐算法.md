# 抖音推荐算法（官方·创作者学院5讲）

> 来源：创作者学院「审核与推荐规则」专题 推荐算法5讲 ｜ 获取：2026-06-16
> 官方口径。喂「选题」「文案」「发布包」。

## 1. 推荐算法原理
算法从全平台视频里，挑"该用户最可能喜欢的"。靠三类数据：
- 视频本身（标题、内容）
- 发布后交互（**点赞、评论、观看时长**）
- 用户历史（看过什么、交互过什么）
→ **让算法推你 = 让对的用户喜欢你**。算法把你的视频推给"喜欢你这类内容的人"。

## 2. 推荐页·上热门四点（官方建议）
推荐页看：点赞率/评论率/**观看时长** + 相似用户喜好 + 创作者粉丝粘性。
1. **持续稳定发布**（帮算法摸清你的潜在观众）
2. **高质量**（多点赞/评论/分享 + 拉长观看时长）
3. **独特创作风格**（提粉丝粘性）← 对应套路库§9 账号风格鲜明
4. **读评论改进**（也呼应"引导互动"）

## 3. 关注页·涨粉
关注页个性化排序，曝光取决于粉丝量。涨粉两招：
1. **真诚告诉观众"关注你能得到什么"**（更新节奏/价值承诺）← 文案 CTA 可加
2. **为粉丝持续创作他们要的**（粉丝粘性越高越易涨新粉）

## 4. 同城
同城靠**封面+标题**选择（交互模式和推荐页不同）。两招：①有意义的封面+标题 ②本地相关内容。
> 迪迪是垂类AI号，同城价值低；但"封面+标题决定点击"这条普适——发布包的封面/标题要下功夫。

## 5. 搜索算法（SEO，迪迪要重视）
搜索匹配"用户需求↔视频"。三类数据：视频可见文字+清晰度+内容 / 用户查询词(NLP) / 互动数据。排名两点：
1. **好视频**
2. **加精准文字描述**（便于搜索理解）← **迪迪发布包：标题/字幕/标签里埋准关键词**（AI agent/Codex/vibecoding/AI创业），吃搜索长尾流量
打压作弊行为。

## 喂给谁（落地动作）
- **选题/文案**：观看时长(完播)是推荐核心 → 北极星对齐；持续稳定发布 + 独特风格 + 引导互动。
- **发布包**：①封面+标题决定点击，必须用心 ②标题/标签/字幕埋精准关键词吃搜索流量 ③CTA 点明"关注我能得到什么"。
- 印证我们方向：官方把"观看时长/粉丝粘性/独特风格/互动"列为核心，跟北极星(完播+涨粉) + 套路库(风格鲜明+引导互动)完全一致。
