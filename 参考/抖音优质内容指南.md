# 抖音官方《精选优质内容指南》提炼

> 来源：抖音作者助手官方视频（item 7529857225054063913）｜ 获取：2026-06-16
> 官方口径，优先级高于我凭经验总结的套路。冲突时以此为准。

## 一句话核心
> **能给用户留下长久美好感受的内容** = 抖音精选/推荐的标准。

## 四大支柱（精选作品的共同特征）

### 1. 获得感（价值）
- 价值导向积极、合规（符合法律 + 社区公约，不引导平台禁止/不鼓励行为）← **这条也是审核红线**
- 内容专业、有利他性、真实有依据（体现该领域专业水准）
- 带入用户视角：满足好奇 / 解答困惑 / 提供帮助
- 激发主动思考：独特视角、新奇观点、开放式提问 → 看完仍有回想
- **迪迪用**：build-in-public 要给"可执行的干货"（对应我们的高收藏逻辑）；结尾认知金句 = 激发思考。

### 2. 惊喜感（与众不同）
- 选题与众不同：反差、不寻常的体验/知识
- 实现过程独特：反常规方式完成某事
- 表现手法创新：音乐/视觉/剪辑/特效组合出新
- **迪迪用**：选题评分卡的"独家性"维度 = 惊喜感；钩子的"结果反差"= 惊喜感。

### 3. 表达力（讲明白）
- 表意精准、逻辑清晰连贯，避免信息冗余/模糊
- 深入浅出：通俗生动，**即使外行也能看明白**
- 风格鲜明：选题视角/视觉/个人特质形成账号记忆点
- **迪迪用**：对应套路库§3"类比讲人话"；"风格鲜明"提醒迪迪固化自己的口吻/视觉记忆点。

### 4. 感染力（共鸣）
- 选题有接受度：大众有感知、愿参与、愿分享
- 讲好完整故事：完整情节让用户沉浸
- 引导用户互动：设置共鸣问题、及时回复评论激发表达
- **迪迪用**：转型故事/第一人称经历 = 完整故事 + 感染力；**这条提醒我们补强"引导互动"**（套路库原来偏收藏，互动这块弱）。

## 与套路库的对照
- ✅ **印证**：我们的"高收藏=干货""独家性""类比讲人话""认知金句"分别对上 获得感/惊喜感/表达力。
- ➕ **官方补充了我们偏弱的**：①**感染力/引导互动**（设共鸣问题、回评论）——套路库原来重收藏轻互动，要补；②**账号风格鲜明**（统一口吻/视觉记忆点）——迪迪要刻意固化。
- 🔴 **审核**：获得感里的"合规/不引导禁止行为"是官方红线，已同步给「预审」。

## 喂给谁
- 选题/文案：把"四大支柱"当**质量自检清单**（一条好内容应同时占到多条）。
- 预审：合规那条进红线。
- 套路库：补"引导互动"和"账号风格鲜明"两块。
