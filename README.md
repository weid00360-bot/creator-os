# 新媒体 Agent · AI 内容创作套件

> VibeCoding 大赏参赛作品 ｜ Build in Public
> 
> 一套跑在 Claude Code 上的 AI agent，帮我把"做一条抖音视频"这件事从头到尾自动化——选题打分、文案起草、发布前预审、发布后复盘，形成数据驱动的闭环。

---

## 它在解决什么问题

做自媒体最贵的成本是**判断力**：选什么题不扑、开头怎么写不被划走、发完怎么知道哪里出了问题。

这套 agent 的目标不是替我"创作"，而是把我自己沉淀的判断标准固化成工具，让每次出手都有依据、每次复盘都能沉淀回来。

---

## 四个 Skill（核心交付件）

| Skill | 触发词 | 干什么 |
|---|---|---|
| **选题** | `/选题` | 拉今日 AI 热点 + 读套路库 → 五维评分卡打分 → 输出排序清单 |
| **文案** | `/文案` | 按套路库结构 + 柱子哥口吻 → 口播稿草稿 + 发布包（标题/tag/封面/预测卡） |
| **预审** | `/预审` | 对照官方红线查限流/违规/标签禁区 → 红黄绿三级清单 |
| **复盘** | `/复盘` | 跑 `loop_review.py` 取后台数据 → 北极星判定 + 留存归因 + 限流预警 |

每个 Skill 的完整 prompt 在 [`skills/`](skills/) 目录。可以直接 fork 去 Claude Code 里装。

## 套路库（知识库核心）

[`爆款知识库/套路库.md`](爆款知识库/套路库.md) — **这是整个 agent 最值钱的东西，不是代码。**

把 27 条同赛道爆款拆成六个维度（选题公式 / 钩子 / 中段结构 / 收尾金句 / CTA / 反面案例），提炼成可复用的规则，喂给 agent 当判断依据。

核心洞察：**收藏率 = 赛道匹配器**。藏/赞 > 0.7 才是精准变现受众，点赞高但收藏低是泛流量陷阱。

[`爆款知识库/拆解卡片/`](爆款知识库/拆解卡片/) 是 27 条爆款的原始拆解，知识库的"原材料"。

## 脚本（自动化层）

```
scripts/
├── loop_review.py    # 复盘取数：自动拉后台北极星指标，生成复盘卡
├── douyin_hot.py     # 抖音热榜 AI 过滤：筛出与 AI 赛道相关的热词
├── monitor.py        # 对标监控：抓指定对标博主的新视频数据
└── douyin_ripper.py  # 单条视频数据 + 逐字稿抓取（用于拆解爆款）
```

取数依赖抖音创作者后台 API（需要 `~/.douyin_cookie`），其余无外部依赖。

## 账号北极星

[`账号北极星.md`](账号北极星.md) 是整套 agent 的总策略文件——定位、受众、指标、禁区、平台分工全在这里。

**核心指标：不看播放量，只看涨粉/千播（目标 >1）和完播率（目标 >5%）。**

## 快速上手

```bash
# 1. clone
git clone https://github.com/YOUR_USERNAME/xinmeiti-agent.git
cd xinmeiti-agent

# 2. 在 Claude Code 里安装 Skill（以选题为例）
# 把 skills/选题/SKILL.md 的内容注册进你的 Claude Code skills 目录

# 3. 跑选题
# 在 Claude Code 里输入 /选题

# 4. 写文案
# /文案 [选题名]

# 5. 发布后复盘（需要抖音 cookie）
python3 scripts/loop_review.py --latest
```

详细配置见 [`工作流SOP.md`](工作流SOP.md)。

## 设计理念

**工作流不值钱，知识库才值钱。**

任何人都可以用一句话让 AI 复刻别人的工作流。真正值钱、别人爬不走的，是你自己沉淀的判断标准——哪类题会爆、哪类钩子留人、什么数据说明选题不行而不是执行不行。

这套 agent 的护城河是套路库和北极星，不是那几段 prompt。
