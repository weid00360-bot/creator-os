# 账号北极星（v1）

> 这是整个新媒体 agent 的总纲。选题、文案、预审、复盘所有环节都必须回到这份文件对齐。
> 最后更新：2026-06-15

---

## 一句话定位

> 一个普通人用 AI 从 0 搭各种 agent / 自动化，把过程和结果讲明白——给想用 AI 创业、做副业、提效的人看。

- 载体：**迪迪（探索世界版）转型**——从 AI 变美号转为 AI 工具/创业号。
- 人设衔接：「探索世界版」天然能装下"探索 AI / 探索创业"，转型话术不别扭。

---

## 目标受众

**想用 AI 搞钱/提效的人**：独立开发者、想转型的职场人、小创业者。

> ❌ 不再是变美消费者。变美粉是上一阶段的资产，与本目标不匹配，转型期接受其流失。

---

## 北极星指标（对 analysis 的正式纠偏）

**不看播放量、不看总涨粉。** 看：

1. **涨粉 / 千播**（受众匹配度）——目标持续 > 1
2. **完播率**——先冲过合格线 5%，再奔 15%+

> 原因：对变现号，一个精准的工具圈粉丝 > 一百个变美路人粉。
> 工具内容在抖音的特性是「**低播放、高精准**」——它是精准粉筛选器，不是流量机器。用对的指标看，它就成立。
> 旧 analysis agent 优化的是播放/涨粉（流量逻辑），与本目标方向相反，复盘时以本指标为准。

## 变现路径

**核心模型：免费公开一个真 agent 引流 → 私域承接 → 付费。**

- 形态：付费社群 / agent 访问权 / 咨询 / 定制（具体产品后定，但内容要为此铺垫）。
- 红线：**卖执行，不卖课。** 见下方禁区。

---

## 内容支柱（按优先级）

1. **Build in Public**（护城河）——我搭了什么 agent、怎么搭、踩了什么坑。真材实料，别人抄不走。
2. **AI 工具实测 / 横评**——帮受众省选型时间。历史数据证明此类涨粉转化最高。
3. **AI 创业 / 变现思路**——机会、案例、方法。筛高商业意向受众。
4. **把复杂 AI 讲人话**（可选）——破圈拉新。

> ⚠️ 呈现方式铁律：**结果先行**。抖音吃前 3 秒结果冲击。
> 先甩"看我用 AI 做出了 X / 前后对比"，搭建过程只作信任背书。
> 只拍写代码、只讲怎么搭 = 自嗨，完播必死。

## 平台分工与漏斗

**市场重心：中文为主，全球一鱼多吃顺带。**

```
GitHub            信任根基 / 作品集（agent 真的在这跑）—— 中外通用，开发者来验货
   ↑ 互相导流 ↓
短视频引流         主：抖音(中文)   顺带：TikTok / IG Reels(英文，换字幕一鱼多吃)
   ↓                结果型钩子 → 评论区口令 / 主页引导 → 筛精准粉
   ↓
私域承接变现       主：微信 / 社群(中文)   顺带：Discord / 邮件 / Gumroad(全球)
```

- 抖音/TikTok/IG **只负责"用结果把对的人钩进来"**，不负责讲透。
- 讲透、建信任、成交 = **私域 + GitHub** 的事。
- 一条口播 → 抖音 + TikTok + IG Reels，换语言/字幕即可，成本低。

## 禁区 / 红线

- 🚫 **变美内容彻底停**（让算法重新给账号定性）。
- 🚫 **不做"教你用 AI 赚钱"的课程贩子**——那是割韭菜区，会和骗子站一队。定位必须是"我搭了能用的东西"。
- 🚫 纯炫技、无落地的 demo。
- 🚫 标题党、无干货。

## 转型过渡策略

转型 = 重新喂标签，不是简单换内容。

1. 老粉（75-80% 变美女性）会变死粉、不互动，**短期拖累推荐权重——这是必经阵痛，别慌**。
2. **过渡期连发同一支柱内容（别混发）**，把新标签结实喂进算法。
3. 接受老粉流失，盯北极星指标（涨粉/千播、完播率）回升，不盯播放。
