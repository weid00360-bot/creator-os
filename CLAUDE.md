## 可用 Skill（直接触发）
- `/选题` → aihot + 抖音热榜 + 套路库 → 五维评分卡排序清单
- `/文案` → 套路库结构 + 柱子哥口吻 → 口播稿草稿 + 发布包
- `/预审` → 按官方红线查限流/违规/标签禁区 → 红黄绿清单
- `/复盘` → 跑 loop_review.py → 北极星判定 + 留存归因 + 限流预警

## 关键规矩
- **文案只存定稿**：草稿在对话里碰，用户说"定稿"才写入 `文案/`
- **人工卡点不替代**：选题拍板 / 文案改口吻+过红线 / 套路库更新 → 必须迪迪决定
- **复盘防过拟合**：单条不下结论，同类攒2-3条再改库；数据24-72h才稳
- **限流先排查**：先查可见性/标签/红线，别归因内容（否则污染套路库）
- **官方口径优先**：`参考/` 里官方规则 > 经验套路
