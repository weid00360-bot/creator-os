## 口播稿（定稿）

**【钩子】**
选题好不容易定了，你有没有过这种时刻：坐那儿发懵——这题到底该用什么风格写才火？想对标某个博主吧，照着写，出来的东西又总差那么点意思。今天我就讲讲，我的文案 agent 怎么把这俩病一起治了。

**【全景带过·1 秒】**
还是先两秒带一下我整套系统：从选题到复盘，5 个 agent 串起来的。上期我拆了选题层，今天讲下一环——文案层。

**【痛点·3 条】**
先说大部分人用 AI 写文案，卡在哪。
第一，题定了，却不知道用什么风格写才火。讲干货还是讲故事？冷静拆解还是情绪带动？全无头绪。
第二，想对标一个博主，照着写却总差点意思——即使原文复刻，也没感觉。
第三，AI 写作没人味，不像人说的话，刷一眼就知道是 AI 干的。

**【三招】**
那我这个文案 agent 怎么解？四招，一个一个看。
**第一招，按选题蒸馏对标博主。** 题一定，它就自动把这个选题底下最火的几个博主全蒸馏一遍——钩子、结构、风格套路全拆出来，直接告诉我：这题，这么写会火。〔上蒸馏成果卡〕你现在看到的这张卡，就是它拆一个头部博主拆出来的，7 步骨架明明白白，这就是我的结构弹药，不用再从零憋。
**第二招，蒸馏自己。** 它会啃我过往所有文案，慢慢长出我的行文风格——我爱用哪个词、句子怎么忽长忽短、毛边留哪儿，它都记着，写出来就像我自己写的。而且它一直给我攒金句，每条结尾那句认知狠话，我基本不用愁。
**第三招，人味儿 skill。** 前面拿了别人的套路、套了我的风格，但 AI 写完那版，我绝不直接用。我有套"人味儿"的标准：从初稿到改稿到事后清单，都聚焦在"具体的人，站在具体的位置说话"——宁愿白描，也不要漂亮话。
**第四招，写完直接配齐发布包 + 预测卡。** 正文写完还不算完——它顺手把标题、标签、封面文案都配好；再出一张预测卡，发布前就先估这条的收藏率、完播大概什么水平。等于我还没发，心里就有数：这条值不值得发、能不能火。等发出去，再拿这张预测卡跟真实数据对账——那是下期复盘的事了。

**【最值钱 / 升华】**
所以它最后产出的，是我的判断、我的话，不是一篇谁都能写的正确稿。而且你发现没有——选题一定，它立马知道这题该怎么写。选题和文案，是连起来的，中间不断档。这就是我上期说的，一套系统该有的样子。

**【CTA】**
这整套 agent，我还是全开源在 GitHub 上，感兴趣自己装来玩，几乎不花钱。想要这套"按选题蒸馏 + 学你风格 + 人味儿把关 + 一键配齐发布"的文案打法，评论区扣个"文案"，我拉你进群，手把手帮你调。

**【金句收尾】**
最后撂一句：AI 能帮你把话说顺，但说不了你心里那句非说不可的话。能写出人味的，从来不是工具，是工具背后那个还愿意较真的人。我是迪迪，下期见。
