## 完整口播稿（定稿）

**〔钩子〕** 这个 skill 能帮你省掉 50% 的 token 用量，让 AI 少说废话；它就是拥有 GitHub 5600 stars 的 skill——Claude token efficient。这条视频，从多能省到怎么装，只要 2 分钟，看完你就能用上。

**〔图表·先上硬货〕**〔画面：挂 benchmark 的省 token/省钱图〕 先看实测：输出 token 砍掉一大截，越贵的模型省得越狠——opus 那档，一轮测试从四毛降到三毛五，输出直接少了六成。单看一次不起眼：让它 review 一段代码，120 个词砍到 30 个词；让它讲个概念，180 词压到 65 词。用得越多，省得越狠。

**〔怎么做到的〕** 它咋做到的？特简单，就一句话：逼 AI 别废话。它给 AI 立了几条铁规矩——比如说开头不准寒暄，别上来"好问题，我很乐意帮你"；等等等，就是把 AI 那些讨好你、凑字数的毛病全改了。最后成果就是：让它看段代码，原来要 120 个词回复，装上之后 30 个词，直接告诉你哪行有 bug、怎么改。省了 75% 用量。

**〔适用范围·一句〕** 但有一点你得知道，这套 skill 不适合所有人，因为这套规矩文件本身也占 token。它只适合经常用 AI 跑项目的（跑 agent、批量写代码、搞自动化，还用贵模型），如果你是平时就偶尔问一两句 AI，那完全没必要装。

**〔怎么装·手把手〕** 最后，怎么装呢？超级简单——把这个 GitHub 网址直接甩给你的 AI（Claude Code、Codex 都行），跟它说"以后但凡跑项目，都按这套原则来"，1 分钟就可以装完。地址我发评论区了，也可以截图保存——欢迎大家今晚就装上试试。我是迪迪，下期见。
