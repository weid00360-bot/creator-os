## 完整口播稿（定稿）

**〔钩子〕** 你现在看到的这条，从选题到剪辑，——全是我自己搭的一个 agent 干的。但今天我不想像卖课的一样说这套自动化流程有多厉害，我想讲点更值钱的：一个真正能用的 AI agent 到底怎么搭。一共有 5 个元素，缺一个都跑不起来。好，那我们开始。

**〔总览·框架〕**〔画面：五要素闭环图〕 在我一块块拆之前，先把整套框架给你看一眼。你看——从我的目标出发，到知识库、到它怎么判断、到整条流程、最后闭环回来，这五块咬在一起，才是一个能自动转、还越转越准的 agent，少一块都不行。下面我把每一块的逻辑和内容完整讲一遍，你跟着我的视频，自己也能照着搭。

**〔①明确目标〕** 第一样，也是最重要的，明确目标——这个 agent 到底要帮你实现什么目的？这个我真交过学费。我这号本来做 AI 变美，出过百万播放的爆款，后来接触了太多 AI 工具、AI 创业，就两边混着发，结果完播率和粉丝狂掉。所以做这个 agent 的时候，我没着急让 AI 干活，而是拉着它一块儿对谈、捋目标。它跟我说：得以终为始——先想清楚这号靠什么赚钱，再倒推回来。这么一聊我才发现，我以前关注的全错了，真正该问的是——钱，从哪儿来？顺着拆：要赚钱 → 得有愿意为 AI 付费的人 → 那就不能要看热闹的泛流量，得要有工具需求、有创业意愿、有点技术认知但又缺真正技术背景的那群精准的人。我把"北极星"定死：这 agent 盯的账号，只服务想用 AI 搞钱的人；数据不看播放量，只看涨粉率和完播率。

**〔②完整知识库〕** 第二样，知识库。这块我特别想多说两句，大部分人都搞反了。很多人以为搭 agent 难在写代码、搭工作流——其实现在工作流根本不值钱，不用花钱买服务，你能下载个 codex 就能跑。一句话——工作流谁都能复刻，真正值钱、别人爬不走的，是你喂给它的知识库。 我怎么建的？把这赛道的爆款，一条条拆成六个维度：选题、开头钩子、文案结构、能复用的套路、金句、数据表现。为啥是这六个？因为它们正好对上"我要做一条视频"必须回答的六个问题——讲什么、怎么开头抓人、怎么组织、能学什么、怎么收尾、它凭什么爆。而最关键的是——这库不能全是网上爬的公开资料，得有你自己的判断沉淀进去。啥叫判断？就是你得定义"什么才是好养料"。比如：别看点赞，看收藏率。一条 AI 新闻 61 万赞，收藏只占 13%，那是吃瓜的泛流量；另一条 1 万 6 赞，收藏占到 97%，那才是能吸引会付钱的人的精准内容——知识库要多收后一种，这才是真正喂得壮 agent 的料。

**〔③评判逻辑〕** 第三样，评判逻辑。光有知识库还不够，你得给它一套"怎么判断"的标准，不然它每次都凭感觉给我选题。我给它一张评分卡，五个维度，每一维都堵一个坑：符不符合北极星——防跑题；是不是高收藏干货——防泛流量；是不是我的独家——防被抄；钩子够不够狠——防没人点；蹭没蹭热点——防过时。而且这五维是我照着自己账号定的，不是套谁的现成模板——这才是它越用越懂我的原因。 落到实际：比如「让 5 个 AI 比赛写文案」这种题，听着挺好玩，一打分——泛流量、收藏零，直接毙；换成「拿刚出的 Kimi 和 Claude 实测搭 agent」，独家加干货加蹭热点，9 分，留。会判断的 agent，自己就把垃圾题挡了。

**〔④清晰流程+人工审核〕** 第四样，清晰的流程，加人工审核。这套 agent 完整链路是：热点和数据进来 → 它按评分卡出选题清单 → 我拍板选哪条 → 它写初稿 → 我改成我说话的腔调、过一遍红线 → 它先预测这条数据大概什么样 → 我录口播 → 剪辑 → 发布 → 数据回流。这里的"剪辑"又是另一套 agent 接手：录前——给文案 → AI 逐段排（展示方式+风格+谁出素材）→ 人工确认 → AI 出 1-2 张示意图和「拍摄清单」→ 我确认风格；录的过程中——提供口播、备好要给 AI 的录屏/截图；录后——AI 转写 → 按敲定方案搭（逐帧字幕+图形+素材）→ 定稿 → 导出 → 复盘。

但你注意，整条里有两个判断我死活不交给 AI：选题和文案。因为这俩恰恰是 AI 的短板。选题，要不要做、值不值得做，这是判断，AI 没有我的品味和盘算；文案，AI 写出来总有股"AI 腔"，踩没踩平台红线它也拿不准——只有人能拍板。说白了，能自动的全甩给 AI，但 AI 干不了的活，必须人来兜底。

**〔⑤闭环迭代〕** 第五样，闭环和迭代，这决定它会不会越来越聪明。视频发出去，它读我的真实数据。哪条爆了，就把那套路在套路库里加权，下次优先给我；哪条扑了，降权。普通 AI 每次都从零开始，我这个，是跟着我账号一起长的。

**〔最终结果〕** 那它真实跑出来，到底给我什么？我每天一睁眼，它已经把一份打好分的选题清单摆我面前——哪条几分、为什么这么打，清清楚楚；我挑一条，它把口播初稿写好；我录完，剪辑那套几分钟出片。就说这条视频——选题是它从清单里挑的、文案是它写的初稿、片子是它剪的，我干的就俩事：出镜、拍板。说白了，这套 agent 给我的结果就一句话：把一个人能干的活，压成了一个团队的产能。

**〔升华+CTA〕** 所以你看——目标、知识库、判断力、流程、闭环。多数人以为搭 agent 是写提示词，其实是设计这一整套思维。说到底，AI 时代早过了比谁模型更聪明的阶段——现在比的是，谁能在你睡觉的时候把那些又苦又累的活悄悄干完，第二天一睁眼，把一个干净的起点交到你手上，让你直接进到最该动脑子的那一步。 我这套 agent 干的就是这个：它干流程，我守判断和审美。而这两样，恰恰是 AI 替不了、也是你最值得花时间的地方。这套怎么搭的我整理成文档全公开了，想自己搞一个的，评论区扣"1"，我发你。以上就是这期的内容，我是迪迪，下期见。
