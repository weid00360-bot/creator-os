## 钩子方式（前段）
- **身份背书 + 连续性**："上一条用 5 个 agent 运营艺人公司，全网破百万" → 立刻接"我所有高频动作都 agent 化了，今天讲一个具体动作"
- **结果反差**："每天 1 分钟，完成一篇完全不用改的口播稿"
- **排除法否定**："不是随便打开 AI 说帮我写——那种又空又假又不像你"（先否定常见做法，立差异）

## 结构亮点（高收藏的根因）
1. **框架先行 + 拟人化命名**：拆成 5 个 agent，每个起名（雷达/Angle/Writer/Jar/memory）= 记忆点，听众好跟。
2. **每个组件都点"判断 / 这步靠我"**：选题 agent 过滤"值不值得我讲"、角度靠人定、审稿不到 8 分继续改、"账号的味道 AI 不能替我判断"——**判断主题贯穿到底**（= 郑经"多讲判断"的极致执行）。
3. **主动标记"最值钱的部分"**：点名 memory（复盘 agent）"我觉得最值钱"，并讲进化感"带着上次修改继续进化、越来越懂我" → 直接引导收藏。
4. **落地画面感升华**："你一个人也能有一个内容团队，成员不是真人，是一组越来越懂你的 agent。"
5. **认知金句**："把重复的、标准化的、可复盘的交给 agent，把最核心的判断/表达/定夺权留在自己手里"（= 护城河论同源）。
